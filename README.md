AI Powered Aptitude Preparation Portal

A full-stack web platform designed to help students prepare for placement aptitude tests through structured practice, company-based questions, and AI-assisted learning tools.

Problem Statement

Many students preparing for campus placements face difficulties in finding structured aptitude practice resources.

Some common issues include:

Previous company aptitude questions are scattered across multiple websites.
Students cannot easily practice topic-wise tests in a structured format.
There is no system that can generate new aptitude questions similar to company exams.
Students often lack tools to analyze their performance and identify weak areas.

This project aims to solve these problems by creating a centralized aptitude preparation platform where students can practice tests, track performance, and prepare efficiently for placement exams.

Solution

This platform provides an interactive aptitude practice system built using the MERN stack with AI integration.

Students will be able to:

Practice topic-based aptitude tests
Solve company-specific previous questions
Take mock tests with timers
View detailed results and performance analytics
Upload aptitude papers for AI-based question extraction

The system will also automatically store new questions discovered through AI analysis, helping the platform continuously grow its question bank.
